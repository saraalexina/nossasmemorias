# 💌 Nossas Memórias

Um projeto desenvolvido com muito carinho para celebrar o nosso Dia dos Namorados e eternizar a nossa história em código e pixels.

## 🚀 Tecnologias e Arquitetura
- **Front-end:** HTML5 e CSS3 (Design Mobile-First).
- **Interatividade:** JavaScript Vanilla (Animações on-scroll e manipulação do DOM).
- **Hospedagem e CI/CD:** GitHub Pages.
- **Banco de Dados (Mensagens):** Supabase.

## ✨ Features Técnicas e Visuais
- Timeline cronológica responsiva.
- Interface fluida com animações de entrada suaves (Intersection Observer).
